# Project 2 - HTML Foundation Page

# Description
This project is a personal biography webpage. It includes information about me, my degree, my interests, hobbies, and my ultimate goal of working in cybersecurity for the FBI. The page includes an image, table of my hobbies, and navigation links between pages. 

# What I learned
I already knew most of the basic HTML, but this project helped me feel more comfortable using it and VS code.I practiced more with headings, paragraphs, lists, and tables. I also learned how to use the handy validator tool, which helped me fix small errors. 

# Challenges
One challenge I faced was fixing the validator errors and making sure my tags were correct. I also had to make sure my links worked correctly between pages. After testing and fixing, I was able to get everything working properly.
