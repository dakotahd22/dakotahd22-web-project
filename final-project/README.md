# My GreenTech Solutions Website
## Project Description
This project is a website for GreenTech Solutions. The website shares information about 
eco-friendly technologies and sustainability for businesses. My website has a home page, about us page,
products and services page, contact information page, and a blog and resources page. This webpage was created
to be quick, accessible, and easy to navigate for users that are interacting from different devices.

## Instructions for Running the Project
First, open the GitHub Pages website link for the GreenTech Solutions project. Then
once navigated to webpage, use the navigation bar to move back and forth from pages on the website.
Users then will be able to view details and information about sustainability, products and services, 
contact info, and blog/resources from the site.

## Technologies Used 
I used HTML5, CSS3, JavaScript, Mobile Web Development, and Web Accessibility to create this website.
